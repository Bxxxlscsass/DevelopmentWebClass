# DevelopmentWebClass

Aula voltada a desenvolvimento web, faculdade abordando a introdução da programação web,

iniciando o estudo com a introdução do HTML, CSS e javaScrpit 

introdução focada na criação de sites, aplicações e serviços que funcionam 

navegador e dependem da comunicação de dados entre cliente (seu computador) e um servidor 

identificar os erros que está no computador do usuário, no código da página ou na infraestrutura do servidor
